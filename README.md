# pandora_test_framework

Scenario 1:
- Go to www.pandora.com
- Register as a new user
- Assign that user is succesfuly registered and signed-in

Scenario 2:
- Go to www.pandora.com
- Search for "Muse" radio
- Assert that "Muse" radio is found

Tests run multi-threaded.

Techs:

- java
- Selenium Web driver
- maven
- testNG
- page objects
- POI
